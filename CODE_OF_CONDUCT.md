# Code of Conduct

## Our promise

We want this project to be safe and welcoming for everyone. Be kind, patient,
and respectful. Humanitarian work includes people from many places and life
experiences.

## Good behavior

- Give helpful feedback about the work, not the person.
- Respect privacy and different points of view.
- Ask questions when something is unclear.
- Accept corrections and take responsibility for mistakes.

Harassment, threats, hateful language, sexual attention, and publishing private
information are not allowed.

## Reporting

Report community problems to the repository maintainers. Use the private
security-reporting path for anything that contains personal or incident data.
Maintainers may edit or remove content and may temporarily or permanently block
people who break these rules.

This code of conduct is based on the Contributor Covenant 2.1:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
